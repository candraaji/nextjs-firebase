## How to Run

1. Clone the repository:
   ```sh
   git clone https://github.com/candraaji/nextjs-firebase.git
   ```
2. Navigate to the project directory:
   ```sh
   cd frontend-repo/
   ```
3. Install dependencies:
   ```sh
   npm install
   ```
4. Start the development server:
   ```sh
   npm start
   ```
